# 🌪 Wind Power Monitoring Platform

> 풍력 발전단지 통합 운영 및 설비 모니터링 시스템 (운영 및 유지보수 / 단독 수행)

---

## 📌 Overview
- 기간: 2024.04 ~ 현재  
- 역할: Backend / Frontend / DevOps (단독 개발 및 운영)  
- 기술: Spring Boot, MyBatis, MariaDB, JSP, JavaScript, Chart.js, REST API  
- 개발 환경: IntelliJ, Postman, DBeaver
